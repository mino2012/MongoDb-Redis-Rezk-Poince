TP 3 : Redis-Mongo

Réponses aux questions :

--> Quelles sont les limites d’une base données orientées document ?

Possibilitées de requêtes moins importantes (Pas de jointures par exemple).
Nécessite plus de mémoire pour le stockage des noms de clefs.


--> Quelles sont les types de données stockés dans Redis, que peut on faire comme types de requêtes ?

Stocké en RAM, une valeur pour une clef.
Des types de données simples : chaîne de caractères, ensembles, tableaux, listes.

Requêtes possibles : SET GET TTL ZADD SADDMongoDb-Redis-Rezk-Poince
